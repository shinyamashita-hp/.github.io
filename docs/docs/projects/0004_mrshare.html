Mr. SHARE / おシェアさま

exhibition

13/06/11 - 06/16

「シェア」をテーマとしたグループ展。リサーチ結果を展示する空間とシェアをテーマにした作品を展示する空間、二つに会場を分割した。作品の使用風景が透けるよう、作品展示側には単純化した室内空間（コントの舞台）を用意した。
