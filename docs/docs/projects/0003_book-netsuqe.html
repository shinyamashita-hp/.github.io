Book Netsuqe

bookmarker

W22 D360 H5 (mm)

読みかけの本を、ちょっとした引っかかりにつるして置いておくことが出来るブックマーカー。
今読んでいる本を、色んな所に持ち運び、色んな所に引っ掛ける。こうするだけで、読みかけの本を忘れることなくいつでもすぐに読み始められる。
ボタンを使った［紐／ボタン］と釣り針を模した［皮／算盤］の二種類を用意した。
